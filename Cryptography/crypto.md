## Encryption
[Learn Encryption](https://youtu.be/xHaxAYDt75Q?si=xUM76G4zMoldwOnV)

Encryption is the process of converting readable data into an unreadable format to protect information from unauthorized access.

### Public Key & Private Key
Public-key cryptography uses two keys:
- A **Public Key** used for encryption
- A **Private Key** used for decryption

This method is commonly used in HTTPS, secure communication, and authentication systems.

## Hashing
[Learn Hashing]([https://www.youtube.com/results?search_query=hashing+explained+cybersecurity](https://youtu.be/_89y4aPfoZs?si=SFRcFESP_GxOyLKW))

Hashing is the process of converting data into a fixed-length value called a hash.

Unlike encryption, hashing is designed to be one-way and cannot normally be reversed.

Hashing is commonly used for:
- Password storage
- Data integrity verification
- Digital signatures

## John the Ripper

[Learn JtR](https://youtu.be/eH9KLksZgYA?si=NGs2Cx0VYNAzIw51)

John the Ripper is a password cracking tool used to identify weak passwords by testing possible password combinations against hashed values.

It is widely used in:
- Password auditing
- CTF challenges
- Cybersecurity training

## Hydra
[Learn hydra](https://youtu.be/yd9vDyzUM_U?si=PjgwQMxydz7xnFD2)

Hydra is a login brute-force tool used to test usernames and passwords against different network services.

It supports protocols such as:
- SSH
- FTP
- HTTP
- Telnet

Hydra is commonly used in:
- Authentication testing
- Security assessments
- CTF practice environments
