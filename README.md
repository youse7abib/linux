# EYCC Cybersecurity Resources

This repository contains beginner cybersecurity resources collected for participants preparing for CTF competitions and cybersecurity events.

## Important Note

This is **NOT** a complete cybersecurity roadmap or a full learning path.

The purpose of this repository is to provide:

* Quick preparation resources
* Beginner-friendly guides
* Useful tools
* Practice platforms
* Basic concepts commonly used in CTF competitions

It is mainly designed for people who:

* Have an upcoming competition
* Want a fast introduction to cybersecurity categories
* Need organized resources to start practicing quickly

---

## Categories Included

* Web Exploitation
* Cryptography
* OSINT
* Digital Forensics
* Reverse Engineering
* Networking Basics
* Linux Fundamentals

Each folder contains curated resources, tools, guides, and practice materials related to its category.
